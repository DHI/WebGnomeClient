Location Files

The Location File Wizard asks questions to help you set up the spill scenario. The questions depends on the specific region being modeled. If you need help answering a Wizard question, use the help button on the Wizard title bar.

Location Files are useful for planning but should not be used to model real oil spills! Visit the Location Files section of the WebGNOME User Manual to learn more.

Pick from available sites on the map below or the menu on the right to load a Location File.
